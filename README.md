# todo-list-spring

Está é uma aplicação semelhante a uma todo list (lista de tarefas), onde será possível cadastrar tarefas, listar tarefas, buscar tarefas pelo id, editar e excluir tarefas (CRUD).


## Tecnologias usadas

* Java 11
* Spring
* H2 Database
* Swagger (Springfox)
* Heroku
* Postresql
